# mycli

- 페이지 링크: https://github.com/dbcli/mycli

## 소개

mycli는 MySQL, MariaDB, Percona를 위한 커멘드라인 인터페이스 입니다. pgcli처럼 자동완성기능과 syntax highlighting 기능을 제공합니다.

## 설치

파이썬을 설치한 후
```
$ pip install mycli
```

OS X 사용자의 경우
```
$ brew update && brew install mycli
```
간단한 명령어 실행으로 설치하면 됩니다.
