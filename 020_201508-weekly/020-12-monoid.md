# Monoid
  - 페이지 링크 : https://github.com/larsenwork/monoid

Monoid는 Coding Font입니다. Font는 이것저것 설명보다 보는게 더 빠르죠~    
바라봤을 때 왼쪽이 적용 전 오른쪽이 적용 후 입니다. 적용전 폰트는 Atom 기본 폰트입니다.
