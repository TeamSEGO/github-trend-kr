 # yapf

Python 개발자에겐 숙명이 있습니다.

PEP8 이라는 코딩 인벤션인데요.

하지만 이 PEP8에는 맹점이 하나 있습니다.

```python
x = {  'a':37,'b':42,

'c':927}

y = 'hello ''world'
z = 'hello '+'world'
a = 'hello {}'.format('world')
class foo  (     object  ):
  def f    (self   ):
    return       37*-+2
  def g(self, x,y=42):
    return y
  def f  (   a ) :
    return      37+-+a[42-x :  y**3]
```
이런 코드도 허용한다는 점이죠..

그래서 **YAPF** 는 **보기좋은 코드**를 위한 코딩인벤션을 제공합니다.

아래와 같이 말이죠

```python
x = {'a': 37, 'b': 42, 'c': 927}

y = 'hello ' 'world'
z = 'hello ' + 'world'
a = 'hello {}'.format('world')


class foo(object):
  def f(self):
    return 37 * -+2
  
  def g(self, x, y=42):
    return y


def f(a):
  return 37 + -+a[42 - x:y ** 3]
```
 - 페이지 링크: [yapf](https://github.com/google/yapf)
